# flutter_application_1


## 项目结构

```
lib/
├── main.dart              // 应用入口
├── app.dart               // App 根组件
├── routes/
│   └── app_routes.dart    // 路由表
├── pages/                 // 页面
│   ├── home/
│   │   ├── home_page.dart      // 首页
│   └── state_examples/         // 状态管理示例
│       ├── self_managed_page.dart    // Widget自管理状态
│       ├── parent_managed_page.dart  // 父组件管理状态
│       └── mixed_state_page.dart     // 混合状态管理
├── widgets/               // 通用组件
│   └── common_button.dart      // 通用按钮组件
├── services/              // 网络 / 本地存储服务
└── models/                // 数据模型
```
